# <center> Financial API - Java </center>

###### <center>API de gerenciamento de clientes e empréstimos</center>

## Tecnologias utilizadas:

- Java 17
- Spring Boot
- Git
- H2 Database
- Maven

## Como executar a API:

1. Primeiramente certifique-se de ter o <strong>Maven</strong> e o <strong>Java JDK 17</strong> instalados em
   sua máquina para poder utilizar seus comandos e rodar a API;

2. Após clonar a aplicação, o Maven fará o download de todas as dependências automaticamente. Lembre-se que, por padrão, a aplicação irá rodar na porta 8080;

3. Você poderá realizar as requisições via Postman ou qualquer outro serviço de sua preferência;
## Recursos da API:

Abaixo serão apresentados os recursos que a API fornece e seus respectivos endpoints.
Com a aplicação rodando você poderá:

- Criar cliente;
- Editar cliente;
- Buscar cliente por CPF;
- Buscar todos os clientes criados;
- Deletar cliente por CPF.

Você poderá também:

- Criar empréstimo para cliente;
- Buscar um empréstimo por ID;
- Buscar todos os empréstimos de um determinado cliente;
- Deletar um empréstimo de um determinado cliente.
